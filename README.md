사용 기술 설명:github, vscode
GitHub 활동 요약:사람들에게 노래제공 및 노래 저장기능 제공하여, 페이지에서 간단히 노래를 들을 수 있도록하는 페이지를 구성하는 코드를 vscode에 작성하고 이를 github에 연동시켜 github에서 페이지 구현하도록 함.
페이지 구성 설명:15종류의 노래들, 라이브러리 기능, 검색기능, 라이브러리 내에서의 노래재생 기능.
회고:페이지 코드 작성 마무리 후, 페이지 점검 시 문제점이 생김->페이지 구성이 복잡해 가시성이 부족하고 간편성이 부족함. 이를 해결하기 위해 코드를 수정함.
